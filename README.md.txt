# MediSense AI Android App

This is a **WebView Android app** that wraps my **Streamlit project** for easy installation on Android devices.

---

## 📱 Android Installation

1. Download the APK file from this repository:  
   [Download app-release.apk](app-release.apk)

2. On your Android device:
   - Make sure **Install unknown apps** is allowed for your browser or file manager.
   - Open the APK and install the app.
   
3. Once installed, open **MediSense AI** and it will load the Streamlit app inside the app.

---

## 🔗 Streamlit App (Reference)

You can also open the Streamlit app directly in any browser:

[https://medisense-ai-phpgh9pwvkrlmyh5z6ywlp.streamlit.app/](https://medisense-ai-phpgh9pwvkrlmyh5z6ywlp.streamlit.app/)

---

## ℹ️ Notes

- The app is a **WebView wrapper**, so it requires an **active internet connection** to load the Streamlit app.  
- The **APK is signed and ready to install** on Android devices.  
- If you want to **build the APK from source**, you can extract the project folder from the ZIP and use **Android Studio** or a WebView builder.
