# MediSense AI

This is an **Android app** that opens the **MediSense AI Streamlit app** inside the app.

---

## 📱 Install on Android

1. Download the APK: [app-release.apk](app-release.apk)  
2. Allow installation from unknown sources on your device.  
3. Open the APK → The app will load your Streamlit project.

---

## 🔗 Streamlit App
